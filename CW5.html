Max Sprite Sheet


Requirements

For Coursework 3 of DGA1020, we were required to set up a spritesheet in Unity


Preparation

In preparation for this task, I worked on my sprite sheet using Piskel as I enjoy using this free online editor.

Piskel allowed me to create the animated movements in pixilated style for the character 'Max'.


The movements required were:

  1. Idle Stance
  2. Walking
  3. Running
  4. Jumping
  5. Dying
  6. Falling

Finalising

Finally, I inputted my spritesheet into Unity and adjusted my frames accordingly to be ready for gameplay within Unity.
